# INVERSE-OF-A-MATRIX
## Aim:
To write a python program to find the inverse of a matrix
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:
### Step1 : Import the numpy module as np.
### Step 2: Store the given matrix in A using np.array().
### Step 3: Using np.linalg.inv(A) inverse the a matrix and store it in a variable.
### Step 4: Print the inverted matrix.

## Program:
```
#Program to find the inverse of a matrix.
#Developed by: Shreeshanth R
#RegisterNumber: 212225040411

import numpy as np
A = [[1,0,3],[-1,2,-2],[2,3,-1]]
B = np.linalg.inv(A)
print(B)
```
